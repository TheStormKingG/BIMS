# Mockup Images

This folder contains the UI screenshots for the Landing page carousel.

## File Naming Convention

### Mobile Mockups
- `mobile-1.png`
- `mobile-2.png`
- `mobile-3.png`
- `mobile-4.png`
- `mobile-5.png`

### Desktop Mockups
- `desktop-1.png`
- `desktop-2.png`
- `desktop-3.png`
- `desktop-4.png`
- `desktop-5.png`

## Recommended Image Dimensions

### Mobile Screenshots
- Aspect Ratio: 9:16 (portrait)
- Recommended Size: 1080x1920px or higher
- Format: PNG (preferred for transparency) or JPG

### Desktop Screenshots
- Aspect Ratio: 16:9 (landscape)
- Recommended Size: 1920x1080px or higher
- Format: PNG (preferred) or JPG

## How to Add Images

1. Take screenshots of your Stashway app in both mobile and desktop views
2. Save them with the naming convention above (mobile-1.png, desktop-1.png, etc.)
3. Place them in this `public/mockups/` folder
4. The Landing page will automatically display them in the carousel

## Note

If images are missing, the component will display placeholder content with emoji icons.
